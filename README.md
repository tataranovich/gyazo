# Gyazo

Unofficial client for Gyazo (http://gyazo.com/).

## Install

### From source

Install Curl, File, ImageMagick, Scrot and Zenity before installing Gyazo.

    $ sudo apt-get install curl file imagemagick scrot zenity
    $ tar -xzf gyazo-XXX.tar.gz
    $ cd gyazo-XXX
    $ sudo make install
